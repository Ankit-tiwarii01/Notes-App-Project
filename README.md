# 📝 Notes App Project

A simple and user-friendly Notes Application built using React. This app allows users to quickly add, view, and delete notes in a clean and minimal interface.

## 🚀 Features
- ➕ Add new notes with title and description
- 📋 View all recent notes
- ❌ Delete notes easily
- 🎨 Clean and responsive UI
- ⚡ Fast and lightweight

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6)
- Tailwind CSS / CSS
- Vite (for fast development)

## 📂 Project Setup

1. Clone the repository:
   git clone https://github.com/your-username/notes-app.git

2. Navigate to project folder:
   cd notes-app

3. Install dependencies:
   npm install

4. Run the project:
   npm run dev
